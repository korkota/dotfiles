# Dotfiles

## Requirements

- Git
- Curl

## Install

Install config tracking in your `$HOME` by running:

```shell
  curl -Lks https://raw.githubusercontent.com/korkota/dotfiles/main/.bin/install.sh | /bin/bash
```
